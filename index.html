Select an image that may contain a blind watermark or hidden data. Use the controls below to inspect different color planes and bit layers.

Tip: Use your keyboard ← and → keys to switch modes once an image is loaded.

About this tool

This viewer lets you inspect the low-order bits (LSB) of an image's color channels. Blind watermarks and simple steganography techniques are often stored in these bits.

Use the mode switches to explore: full-color channels (R / G / B / Alpha), inverted colors, and individual bit planes for each channel (Plane 0, 1, 2).
